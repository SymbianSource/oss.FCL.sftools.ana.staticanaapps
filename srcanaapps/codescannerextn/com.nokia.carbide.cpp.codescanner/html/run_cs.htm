Running CodeScanner

CodeScanner is a static source code analysis tool for Symbian OS programs written in C++. CodeScanner behaves differently depending upon the type of resource selected in the C/C++ Project, Navigator, Project Explorer, or Symbian Project Navigator views:

  • project, folder, or file - scans the project associated with the selected project, folder, or file
  • MMP file - scans the selected .MMP file, plus any user include path(s)/source file(s)/resource file(s) listed in the .MMP file
  • INF file - scans the selected .INF file, plus any .MMP file(s) listed in that .INF file, as well as the user include path(s)/source file(s)/resource file(s) listed in the .MMP file(s)

You can run CodeScanner using either of these methods:

  • Running CodeScanner within Carbide
  • Running CodeScanner from the Command-line
