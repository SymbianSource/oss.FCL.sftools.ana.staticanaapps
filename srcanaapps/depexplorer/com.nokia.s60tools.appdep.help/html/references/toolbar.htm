Toolbar

The toolbar in the Dependency Explorer view contains the following items:

You can access context menus by right-clicking a node or a component in the Dependency Explorer main view, or by clicking the small arrow on the toolbar.

SDK and component management

Button Name When to use?
Select an SDK Opens the SDK Selection wizard page. Use it to change the SDK, build targets, and SIS files you want to analyse.
Select a new root Component Opens the Component Selection page. Use it to change the root component for analysis.
Update cache Updates the cache for your currently selected build targets.
Find Opens the Find Components dialog, in which you can search for components in the hierarchy.
Search Opens the Search dialog which can be used for searching for components, imported functions and exported functions from a selected SDK. The default search scope used for the search dialog is based on the view from which the search action is started.

Component navigation

Button Name When to use?
Expand All Expands the entire hierarchy under a selected node or a component.

Tip: To expland only a specific subtree, select the following from the component's context menu: This option is useful in cases when the overall hierarchy is very long.
Collapse All Collapses the entire component hierarchy.
Go Home Go Home action goes back to the top level in the component tree.

This action is available only if the Go Into action has been performed previously.
Go Back Goes back up one level in the component tree.

This action is available only if the Go Into action has been performed previously.
Go Into Drills into the selected subtree or into the lower levels of the component hierarchy.

This action is only available for the component nodes that have children (marked with:) .
