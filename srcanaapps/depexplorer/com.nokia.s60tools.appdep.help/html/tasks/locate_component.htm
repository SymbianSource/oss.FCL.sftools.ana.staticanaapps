Locating components

To locate components:

  1. Right-click a component in the component tree, (for example hal.dll) from Select an SDK > arm5 Release > inetprotutil.dll > bafl.dll > hal.dll
  2. Select Locate Component... The Select concrete component dialog opens showing all alternatives that contain the hal.dll substring:
    • _h2_hal.dll
    • _h4hrp_hal.dll
    • _integrator_cm1136_hal.dll
    • _template_hal.dll
  3. Select the desired concrete component for hal.dll, and click OK. The component you selected, appears in the component tree. Now you can analyse the component in more detail. For more information, see Analysing components.

    If you are familiar with prefixes, you can use the Open prefix preferences option. Adding a prefix helps you in locating components without manual searching.

  4. Click the Open prefix preferences link if you want to define a prefix or prefixes for the component.
  5. Add or Remove a prefix in the Component prefix search order box, and define the search order using the Up and Down buttons.

  6. Note! You can change the prefix preferences. For instructions, see Setting preferences.

  7. After setting the prefix preference, click Apply > OK.
