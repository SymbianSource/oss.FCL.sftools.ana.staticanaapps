Exporting the report

Dependency Explorer offers a functionality to share the results of the analysis by exporting them as an HTML file. The exported report shows the structure of the component tree and lists the properties information, as well as the exported and imported functions of all the components.

To export a report:

  1. Right-click a component and from the pop-up menu, select Export report...

    The content of the report is generated based on the component that was clicked. If you selected the root component, the report will include the entire tree hierarchy. If the selected component was from a lower level, the report will only include the component tree hierarchy from that component downwards.

  2. In the Export Report dialog, specify the location for the report and click Save.
