Checking component properties

The properties displayed for components are: Directory, Filename, Binary format, Secure ID, Vendor ID, UID1, UID2, UID3, Capabilities, Min and Max Heap Size, Stack Size and DLL ref table count.

To view the properties:

  1. Select the component in the Dependency Explorer view.
  2. Do either of the following:
    • Click the Component Properties tab.

    • Right-click the component and from the pop-up menu, select Properties. The properties are displayed in a separate window:

  3. (Optional) To copy the information to clipboard, right-click the properties and select Copy.
