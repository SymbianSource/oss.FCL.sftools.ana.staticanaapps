Managing SDKs

Before you can start analysing components in Dependency Explorer, install the necessary SDKs to Carbide.c++. For guidelines on what to install, see the recommendations in Software Development Kit (SDK).

  • To install SDKs, go to the Window > Preferences > Carbide.c++ > SDK Preferences > Add New SDK menu, and select the item to be added.
  • To view the installed SDKs, double-click the root node in the Dependency Explorer view:

    The available items are listed in the SDK Selection wizard:

  • To change the SDK being viewed at any time, click on the Dependency Explorer toolbar.
