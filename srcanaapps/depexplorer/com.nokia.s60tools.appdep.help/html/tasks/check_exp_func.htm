Checking exported functions

You can view which functions have been exported from the component. The information about exported functions consists of an ordinal number and a name.

To view the exported functions of a component:

  1. Select the component in the Dependency Explorer view.
  2. Click the Exported Functions tab.

    The functions provided by the component are listed in the tab.

  3. (Optional) To copy the information to the clipboard, right-click anywhere on the list and select Copy.
