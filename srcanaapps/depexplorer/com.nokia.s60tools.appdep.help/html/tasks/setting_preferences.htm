Setting preferences

To set the Dependency Explorer preferences:

  1. Select Window > Preferences... > Carbide Extensions > Dependency Explorer Preferences.
  2. Use the Add or Remove button in defining the prefix in the Component prefix search order box, and define the search order using the Up and Down buttons.
  3. If you check the Confirm set as new root action check box, the component is automatically set as a new root without confirmation whenever the set as new root action is invoked due to double-click event, and the confirmation is not asked anymore.
  4. After setting the prefix preferences, click Apply > OK.
