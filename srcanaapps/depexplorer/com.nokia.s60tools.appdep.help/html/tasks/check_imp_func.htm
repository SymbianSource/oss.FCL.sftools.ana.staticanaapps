Checking imported functions

You can view which functions have been imported to the component. The information about imported functions consists of an ordinal number, a name and an offset value.

To view the imported functions of a component:

  1. Select the component in the Dependency Explorer view.
  2. Click the Imported Functions tab.

    The functions used by the component are listed in the tab.

  3. (Optional) To copy the information to clipboard, right-click anywhere on the list and select Copy.
