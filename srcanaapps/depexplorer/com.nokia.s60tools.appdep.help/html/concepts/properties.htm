Properties

In addition to analysing components, Dependency Explorer can also be used to display the properties information of a component.

The properties displayed include Directory, Filename, Binary format, Secure ID, Vendor ID, UID1, UID2, UID3, Capabilities, Min and Max Heap Size, Stack Size and DLL ref table count.
